## v3 신규/심화 산출물

1. **관세율 변동 시나리오 시뮬레이션(`05`)** — 한 수출 건에 대해 현행(FTA 적용)·FTA 실효·양허 부분복귀(½)·추가관세(+10%p) 4시나리오의 관세·착지원가·마진율을 한 화면에 막대+표로 비교. 목표판매가는 상품가×(1+30% `[추정]`)로 잡아 마진율을 산정한다. MFN 0% 품목은 실효 시에도 관세 0, 추가관세 시나리오에서만 관세가 발생해 도메인 정합적이다.
2. **다국가 공급망 원산지 누적 roll-up(`06`)** — 부품/원재료를 원산지·금액·역내(누적대상) 여부로 트리 입력하면 `RVC = 역내가치/총가치`를 roll-up 계산한다. 수분크림 세트는 국내 원료·용기 + 프랑스 향료(역외) + 베트남 포장(역내 누적) → RVC 90%로 한미FTA 문턱 35%를 +55%p 초과 충족. "누적 RVC를 원산지에 반영" 버튼으로 원산지 판정에 직접 반영한다.
3. **FTA 사후검증 리스크 스코어(`08`)** — 원산지 충족 건의 RVC 마진(얇을수록↑)·C/O 발급유형(자율발급↑)·서류 완비도·역내 누적 의존을 가중 합산해 0~100 리스크를 게이지로 시각화한다. 마진 여유·기관발급·서류 완비 건은 낮음, 자율발급·얇은 마진 건은 보통/높음으로 차등된다.
4. **수출 채산성 최적 협정·국가 추천(`07`·`13`)** — 도착국×적용협정 조합별 `순채산 = 목표판매가 − 착지원가 + 환급`을 계산해 내림차순으로 추천 경로를 띄운다(물류요율·협정 절감·환급 반영). 영문 화면(`13`)에서도 동일하게 동작.
5. **KO/EN 다국어 i18n(`12`·`13`·`10`·`11`)** — 언어 토글(KO↔EN)이 사이드바·모바일 상단·드로어에 노출되고, 대시보드(Broker Workspace·FTA Savings·Drawback Est.)·메뉴·뷰 라벨·PDF가 영문으로 전환된다. **해외 바이어 역할 진입 시 자동 EN** 으로 전환.
6. **해외 바이어 포털(`10`·`11`)** — 해외 바이어가 자신에게 배정된 주문의 상태·영문 서류·ETA·L/C·B/L을 EN UI로 조회한다. "주문·서류 조회" 시 바이어 포털 세션 ID가 연동 로그에 아티팩트로 남고, 영문 PDF를 내려받는다.
7. **무역금융 L/C·네고(`09`)** — 은행 역할이 신고서 준비된 건에 신용장(L/C)을 개설(SWIFT MT700 mock)하고, 이후 선적서류를 매입(네고)한다. 네고 시 수수료 2% `[추정]`를 차감한 지급액이 산정되고 SWIFT/네고 참조번호가 로그에 남는다.
8. **선사 EDI(`15`)** — 포워더 선적 부킹 시 B/L·ETA 발급과 함께 선사 EDI 적하목록(Manifest) 메시지가 mock 송신되어 참조번호가 로그에 남는다.
9. **통합 연동 로그 7채널(`14`)** — UNI-PASS·환급접수·환율API·포워더·무역금융·바이어포털·선사EDI 7개 mock 연동의 전체 전송 이력을 채널 필터로 조회.

## 4. 실행/구동 방법

1. `projects/trade-customs/v3.html` 을 브라우저로 연다 (더블클릭 또는 `file://`).
2. 기본 관세사 역할·KO 로 시드(3개 수출기업 6건)가 표시된다. 사이드바(모바일=더보기 드로어)에서 **언어(KO/EN)·역할(5종)·테넌트**를 전환한다.
3. 워크플로 시연: ① 품목·HS → 가중 분류 → FTA 원산지·최적화 → 신고서 → UNI-PASS 전송→수리 ② 환급 접수→심사→지급 ③ 포워더 부킹→B/L·ETA·선사 EDI ④ (은행 역할) L/C 개설(MT700)→네고 ⑤ 공급망 부품 누적→원산지 반영→사후검증 리스크 점검. 고급 알고리즘(관세율 시나리오·채산성 최적화)은 분류된 건마다 자동 산정된다.
4. 새로고침 후에도 진행 상태·역할·테넌트·언어가 유지된다(localStorage). 연동 로그·설정에서 초기화 가능.
5. 재캡처: `cd projects/trade-customs && node capture-v3.mjs` (워크스페이스 루트 `node_modules` 심링크로 playwright 해석). v1/v2 데모는 `node capture.mjs`/`node capture-v2.mjs` 로 별도 보존.

- 외부 연동 7건(UNI-PASS·환급·환율·포워더·무역금융 SWIFT·바이어포털·선사 EDI)은 **mock 전송 로그**로, 정식 버전은 관세청 전자인증서·은행 SWIFT 망·선사 실 EDI·실 환율 피드로 승격한다.
