# 5_research — 근거자료 (목표 출처 1,000+)

**수집 현황: 12 / 1,000 (정직 표기).** 제안서([`../1_제안서.md`](../1_제안서.md)) 본문 인용에 필요한 **핵심 통계·규제·이론 출처만 우선 확보**했다. 잔여(1,000 목표 대비)는 `research-collector` 병렬 에이전트로 분산 수집해 후속 통합한다. **허위 충족 표기 아님** — 미달을 그대로 밝힌다.

비즈니스 수치(CAC·LTV·전환율·가격·시간절감 등)는 제안서에서 전부 **`[추정]`** 으로 표기했고, 아래 공식 출처와 한 문장에 섞지 않았다.
