## 3. 환경

| 항목 | 값 |
|:---|:---|
| OS | macOS (Darwin 24.6.0) |
| 구동 | 브라우저 직접 열기(`file://`) 또는 정적 서버 — 오프라인 단독 |
| 언어 | HTML5 + 바닐라 JavaScript (ESM 불요, 빌드리스) |
| 라이브러리 | jsPDF 2.5.1 (CDN, 부재 시 mock 폴백) |
| 저장소 | 브라우저 localStorage |
| 캡처 뷰포트 | PC 1280±(데스크톱) · 모바일 390×844 |

## 4. 실행/구동 방법

1. `projects/senior-care/v1.html`을 브라우저로 연다(더블클릭 또는 정적 서버).
2. 최초 진입 시 데모 데이터가 자동 시드된다. 관리자 화면이 기본 표시.
3. 좌측 사이드바(모바일은 하단 탭바/더보기 드로어)로 뷰를 전환한다.
4. 역할 칩으로 관리자·사회복지사·요양보호사·보호자를 전환한다(`?role=guardian` 쿼리로 직접 진입 가능).
5. 케어플랜·방문기록·청구 워크플로를 끝까지 진행하고, PDF 버튼으로 문서를 생성한다.
6. 새로고침해 입력 데이터가 유지됨을 확인한다. 콘솔에 `돌봄노트 v1 · 구현 기능 수: 120`이 출력된다.
