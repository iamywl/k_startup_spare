## 2. 구현/제작 범위

- 단일 HTML(`v3.html`) + 외부 로직(`app-v3.js`) + 폰트 번들(v2 `nanum-font.js` 재사용) — 빌드 불필요, `file://` 오프라인 자체완결.
- **8개 라우트**: 분석 대시보드 / 전자세금계산서 / 연차·근태 정산 / 급여·이체 / 노무사 자문 / 감사 로그 / 다중회사 콘솔 / 전자계약. + KO/EN × 오너/관리자/직원 분기로 **실효 뷰 10종+**.
- **워크플로 4개+**: ① 세금계산서 발급(공급가액→세액→API→발급내역→PDF) ② 급여 정산→펌뱅킹 CSV 생성 ③ 연차 정산→ICS 내보내기 ④ 노무 챗(질의→근거조항→후속질문) ⑤ 권한 전환→메뉴 분기→감사로그 적재.
- **3개 테넌트** 데이터 격리 + 통합 콘솔 합산. 영속 계층 `localStorage[smb_state_v3]`(v2 키와 분리).
- 로그인 없이 게스트 모드(역할 스위처로 RBAC 시연) — CLAUDE.md §3.4.
