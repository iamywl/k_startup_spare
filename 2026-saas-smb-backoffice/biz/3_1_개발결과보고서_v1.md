## 1. 성과품 매핑

| 과업지시서 §5 항목 | 본 v1 산출물 | 충족 |
|---|---|---|

## 2. 구현/제작 범위

- 단일 페이지 SPA (HTML + Tailwind CDN + jsPDF 2.5.1)
- **8개 라우트**: 대시보드 / 전자계약 / 계약 워크플로 / 근태 / 급여 / 직원 관리 / 알림톡 로그 / 문서함
- 4단계 계약 발행 워크플로 (stepper UI, 단계 간 데이터 보존)
- 직원 CRUD + 계약 / 알림톡 영속 (`localStorage[smb_state_v1]`)
- **로그인 없이 게스트 모드** (CLAUDE.md §3.4)

```bash
# (1) 앱 단독 구동 — 외부 인프라 없이 file:// 로 동작
open /Users/ywlee/k_startup_spare/2026-saas-smb-backoffice/projects/smb-backoffice/index.html

# (2) 캡처·검증 재현 (Playwright 일시 설치 후 자체 스크립트 실행)
cd /Users/ywlee/k_startup_spare/2026-saas-smb-backoffice/projects/smb-backoffice
npm init -y && npm i playwright@^1.59.1   # Chromium 캐시 존재 시 추가 다운로드 없음
node capture.mjs                           # 14장 캡처 + PDF/서명/CRUD/영속 검증 출력
```
