# 과업지시서 v3 — 현장 작업자 모바일 협업 SaaS

> v3 위탁 분(가정): **산업별 체크리스트 콘텐츠 대량 확충 + 보험사 위험분석 모델 검증 + 네이티브 래퍼 빌드/스토어 등록 + 안전 KPI 시각화 디자인** 시나리오. 본 v3의 SW 본체 확장(라이브러리 엔진·위험점수 알고리즘·SLA 룰·4역할 권한·오프라인 충돌해결)은 자체 수행한다.

## 1. 과업의 개요

### 과업의 목적

1) v2 5억 베타 검수 통과 후, **시리즈A 데모** 수준으로 격상하기 위한 외주 위탁.
2) 제안서([`1_제안서.md`](./1_제안서.md)) §8 로드맵 v3(M7~12) — "산업별 체크리스트 라이브러리 + 보험사 위험분석 API + iOS/Android 네이티브 래퍼" — 의 실증.
3) v2 한계 — 단일 산업 템플릿(각 4~5항목)·온라인 위험분석 부재·웹 전용 배포·KPI 추세 미시각화·역할 3종 — 을 해소하여 보험사 데이터 제휴([§6.1 수익모델 R3](./1_제안서.md))·대기업 Enterprise 영업의 데모 자산을 확보.

### 과업의 범위

1) 외주 수행 업무 (가정)
   - 산업별 체크리스트 콘텐츠 대량 확충 (제조·F&B·물류 각 다수 세트·항목, 법령·표준 매핑)
   - 보험사 위험분석 스코어링 모델 계수 검증 (언더라이팅 자문·실손해율 데이터 정합)
   - iOS/Android 네이티브 래퍼(Capacitor) 빌드·서명·스토어 등록 대행
   - 안전 KPI 대시보드 차트 디자인 시스템·일러스트
2) 범위 외 (Out of Scope) — 자체 수행
   - 산업별 체크리스트 **라이브러리 엔진**(선택→작업 적용) 구현
   - **위험점수 산출 알고리즘**(이상보고·심각도·체크 준수율·적체 가중 → 점수·등급·권고) 구현
   - **현장 안전 KPI 대시보드**(라인별 사고/이상/완료율 추세) 구현
   - **푸시 캠페인**(예약·발송 로그) 구현
   - **SLA/에스컬레이션 룰 엔진** 구현
   - **역할 4종 권한 분기** 구현
   - **오프라인 큐 충돌 해결(LWW)·동기화 상세** 구현
   - 네이티브 래퍼 **install 안내 UI + manifest/SW 고도화** 구현

## 2. 과업의 세부 내용

- 시제품 사양: 모바일(iOS/Android 래퍼 + PWA) + 데스크톱 반응형. 오프라인 큐·충돌해결, 푸시 캠페인, 위치/카메라 옵트인.
- 위험분석 API: 라인별 입력(이상보고 건수·심각도 가중·체크리스트 준수율·작업 적체·24h 내 중대사고) → 0~100 위험점수·A~D 등급·보험료 영향(±%)·룰 기반 권고. 외부 API 키 부재 시 동일 알고리즘 mock fallback.
- KPI: 라인별 7일 시계열(사고·이상보고·완료율) 스파크라인 + 주간 집계 KPI.
- 체크리스트 라이브러리: 제조·F&B·물류 산업별 세트(예: 제조 LOTO·컨베이어·프레스·화학물질 / F&B HACCP·온도관리·마감위생 / 물류 입고검수·지게차·상차고박)를 작업에 적용·교체.
- 역할 4종: 작업자·반장·PM·**안전관리자**. 역할별 내비게이션·뷰·허용 액션(권한 매트릭스).
- SLA: 우선순위별 한도(분) 초과 시 자동 상향보고 후보 판정 + 에스컬레이션 실행.
- 네이티브: manifest(아이콘·shortcuts·maskable) + SW(push 핸들러 포함) 고도화 + iOS/Android 설치 안내 시트.
- 다국어: v3는 한국어 (영어는 v4 이후).

> **v3 자체 산출물 핵심 8가지** (외주와 별도, 본 사이클 [`5_1_개발결과보고서_v3.md`](./5_1_개발결과보고서_v3.md) §6 검수 대상):
> 1. **산업별 체크리스트 라이브러리** — 제조·F&B·물류 다수 세트(총 10세트·50항목), 선택→작업 적용·교체
> 2. **보험사 위험분석 API 시뮬레이션** — 이상보고·심각도·체크 준수율·적체 가중 → 위험점수(0~100)·등급(A~D)·보험료 영향·권고 산출(실 계산 알고리즘)
> 3. **현장 안전 KPI 대시보드** — 라인별 7일 사고/이상/완료율 스파크라인 추세 + 주간 집계
> 4. **네이티브 래퍼(iOS/Android) install 안내 + manifest/SW 고도화** — 아이콘·shortcuts·maskable·push 핸들러
> 5. **푸시 캠페인** — 전사 안전공지 예약·발송 + 도달·발송 로그
> 6. **작업 SLA/에스컬레이션** — 우선순위별 한도 초과 자동 상향보고 룰 엔진
> 7. **역할 4종+ 권한** — 작업자·반장·PM·안전관리자, 역할별 뷰·허용 액션(권한 매트릭스)
> 8. **오프라인 큐 충돌 해결/동기화 상세** — IndexedDB 큐 + LWW(last-write-wins) 충돌 해결 + 동기화 로그

## 변경 이력 (v2 → v3)

| 구분 | v2 | v3 변경점 |
|:---|:---|:---|
| 체크리스트 | 산업별 단일 템플릿(제조5·F&B5·물류4) | **라이브러리화** — 산업별 다수 세트(10세트·50항목)·작업 적용/교체 |
| 위험분석 | 없음 | **보험사 위험분석 API 시뮬레이션** 신규 (실 스코어링 알고리즘) |
| KPI | 단일 시점 KPI·진척 바 | **라인별 7일 추세 차트(사고/이상/완료율)** 신규 |
| 배포 | PWA(SW+manifest) | **네이티브 래퍼 install 안내 + manifest/SW 고도화**(shortcuts·push 핸들러) |
| 알림 | 푸시 구독 | **푸시 캠페인**(예약·발송 로그) 신규 |
| 룰 엔진 | 없음 | **SLA/에스컬레이션 룰 엔진** 신규 |
| 역할 | 3종(작업자·반장·PM) | **4종**(+안전관리자), 권한 매트릭스 |
| 오프라인 | IndexedDB 큐 drain | **LWW 충돌 해결 + 동기화 상세 로그** |
