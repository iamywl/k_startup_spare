# 과업지시서 v2 — 현장 작업자 모바일 협업 SaaS

> v2 위탁 분: **PWA 인프라 + 디자인 시스템 v2 + UX 라이팅** 가정 시나리오. 본 v2의 SW 본체 확장은 자체 수행.

## 1. 과업의 개요

### 과업의 목적

1) v1 1천만원 PoC 검수 통과 후, **5억 베타 출시** 단계로 격상하기 위한 외주 위탁.
2) 토스·카카오뱅크 수준의 모바일 UX, 데스크톱 동시 지원, 오프라인·푸시·디바이스 API 통합으로 *프로덕션* 수준 도달.
3) v1 한계 — 단일 라인·단일 사용자·온라인 한정·디자인 일관성 부족 — 을 해소.

### 과업의 범위

1) 외주 수행 업무
   - 디자인 시스템 v2 (라이트/다크·WCAG AA·터치 56pt+ 가이드)
   - 빈 상태/에러/로딩 일러스트 30종
   - UX 라이팅 (200 카피·한국어 존댓말 톤)
   - 산업별 체크리스트 콘텐츠 (제조·F&B·물류 각 30개 항목)
2) 범위 외
   - PWA 인프라(IndexedDB·Service Worker·Push) 구현 — 자체
   - 카메라·위치·진동 등 디바이스 API 통합 — 자체

## 2. 과업의 세부 내용

- 시제품 사양: 모바일 + 데스크톱 반응형 PWA, 오프라인 큐 + 푸시 + 카메라/위치 옵트인
- UI/UX: 토스·카카오뱅크 톤 — 큰 타입·둥근 모서리·소프트 섀도·Bottom Sheet
- 콘텐츠: 산업별 체크리스트(제조 30 + F&B 30 + 물류 30) JSON 포맷
- 다국어: v2는 한국어만 (v3 영어)

> **v2 자체 산출물 핵심 8가지** (외주와 별도, 본 보고서 §6 검수 대상):
> 1. 모바일+데스크톱 반응형 (Tailwind sm/md/lg/xl)
> 2. 다중 사용자/다중 라인 시뮬레이션 (역할 3종: 작업자·반장·PM)
> 3. IndexedDB 기반 오프라인 큐 (offline 표시 + 복구 시 sync)
> 4. Service Worker (PWA install 가능)
> 5. Web Push API mock 권한·구독·표시
> 6. Camera Stream (getUserMedia) 실 카메라 캡처
> 7. Geolocation 작업 위치 자동 기록
> 8. 산업별 체크리스트 적용 화면
