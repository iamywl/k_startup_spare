# 과업지시서 v1 — 현장 작업자 모바일 협업 SaaS

> 외주 위탁 분: **UI 디자인 가이드** (가정 시나리오). 본 v1의 SW 구현 본체는 자체 수행한다. v2에서 추가 외주(예: 음성 인식 백엔드 튜닝) 발생 시 별도 문서로 분리한다.

## 1. 과업의 개요

### 과업의 목적
1) 시제품 모바일 PWA 의 시각적 일관성·접근성을 갖춘 디자인 가이드를 외주 디자이너로부터 확보한다.
2) 자체 팀이 보유하지 못한 모바일 UI 전문성을 외주로 보완하여, 제안서([1_제안서.md](./1_제안서.md))의 "현장 워커 모바일 퍼스트" 차별점을 시각적으로 입증한다.

### 과업의 범위
1) 외주가 수행할 업무
   - 컬러 팔레트·타이포그래피·아이콘 세트(SVG) 제공
   - 큰 버튼·고대비·장갑 끼고 누를 수 있는 터치 영역 가이드(최소 56pt)
   - 작업지시 카드·이상보고 폼·완료 인증 모달 시안 (Figma)
2) 범위 외 (Out of Scope)
   - 코드 구현 일체 (자체 수행)
   - 백엔드·API·DB 설계
   - 운영·유지보수

## 2. 과업의 세부 내용

- 타깃 사용자: 제조 라인 작업자 (만 30~55세, 남성 우세, 안전화·장갑 착용 환경)
- 디자인 톤: 산업용·고대비·저자극. 채도 낮은 색조 + 강조색 1개.
- 화면 시안 5개: ① 작업지시 목록 ② 작업지시 상세 ③ 완료 인증 ④ 이상보고 폼 ⑤ 관리자 대시보드
- 다크 모드 미요구 (현장 조명 환경 고려)
