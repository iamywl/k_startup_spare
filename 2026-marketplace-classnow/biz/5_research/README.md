# 5_research — 근거자료 (목표 출처 1,000+)

> `CLAUDE.md` §2.5·§2.6. **실제 존재하는 출처만**(날조·중복·유령 인용 금지). 진행 중이면 현재/목표 수치를 정직히 명시한다.

**수집 현황: 30 / 1,000** — 제안서 v1 본문이 직접 인용하는 1차 근거를 우선 확정했다. 나머지는 §9 병렬 `research-collector` 로 KOSIS 표 단위·부처 보도자료·KCI 논문·국제기구로 확장 예정(허위 충족 표기 금지).
