# car-advisor

연봉 기반 차량 옵션 추천 + 예상 유지비 데모 앱 (standalone, 백엔드 없음).

## 구성

| 파일 | 설명 |
|---|---|
| `index.html` | v1 — 추천·옵션 필터·유지비·주행거리 |
| `v2.html` | v2 — 전기차 보조금·실구매가·5년 TCO·KNCAP/기계식주차·프로필·동기화·ⓘ 툴팁 |
| `v3.html` | v3 — 차별화 17종(라이프스타일 마법사·월 부담률·지자체 보조금 비교·EV vs 가솔린·구매방식·부정검색·실연비·공유 카드 등) |
| `data.js` | 차량 데이터셋(31대) + 옵션·페르소나·동의어·유지비·보조금·후기·패키지 상수 (`window.*` 전역, v1~v3 공용) |
| `capture*.mjs` | 버전별 실 구동 캡처 스크립트 |

## 실행

```bash
# 그냥 브라우저로 열기 (오프라인 구동)
open index.html         # macOS

# 캡처 재생성 (workspace 루트에서)
npm i playwright
node 2026-car-salary-advisor/projects/car-advisor/capture.mjs
```

## 기능

- 연봉·연간 주행거리·예산 성향·우선순위 입력 → 감당 가능 차량 추천·정렬
- 패싯 필터(유종·세그먼트) + 체감 옵션 칩 5그룹 + 페르소나 태그 + 동의어 검색(엉따/360/차박…)
- 예상 유지비(연료·자동차세·보험·정비) 분해 + 주행거리 슬라이더 즉시 재계산
- 차량 비교함, localStorage 영속
