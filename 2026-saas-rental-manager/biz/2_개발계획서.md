## 현재 상태
- v1 완료. 5개 뷰 + 다단계 워크플로(호실→임차인→청구→수납/독촉→영수증) 실 동작.
- PDF(계약서·영수증) 실 생성·다운로드 확인, 사진 업로드/미리보기 동작, localStorage 지속 확인.
- 캡처 PC 11장 + 모바일 12장, 반응형 눈검수 통과(사이드바 하이재킹·차트붕괴·overflow 없음).
- 블로커 없음.

## 기술 스택
HTML 단일 파일 / Tailwind(CDN) / Chart.js 4 / jsPDF 2.5 + html2canvas 1.4 / Pretendard / Playwright(캡처용, 커밋 제외).
