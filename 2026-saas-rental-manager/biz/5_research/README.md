# 5_research — 임대지기 조사·근거자료

> 본 폴더는 [`../1_제안서.md`](../1_제안서.md)가 인용하는 통계·근거의 통합 보관소다.
> 공공 통계 포털 → 정부 부처 → 국책연구원 순으로 출처 우선순위를 둔다.
> 확정 수치는 각주로 출처를 명시하고, 검증이 끝나지 않은 값은 본문에서 `[추정]` / `[재확인 필요]`로 표기한다.

## 1. 자료 목록

| 파일/출처 | 발행 | 요약 | 본문 용도 |
|:---|:---|:---|:---|
| 통계청 「인구주택총조사」 점유형태 집계 | 2020~ | 전국 가구의 약 38~39%가 임차(전세·보증부월세·월세) 가구 | Problem §시장규모 |
| 국토교통부 「주택 임대차 신고제(전월세신고제)」 운영 자료 | 2021 시행~ | 보증금 6천만원·월세 30만원 초과 임대차 신고 의무화 → 임대인 행정부담 증가 | Problem §규제 트리거 |
| 통계청·한국부동산원 「임대차 거래량/전월세 통계」 | 월간 | 전월세 거래가 매매 대비 다수를 차지하는 임대차 중심 시장 구조 | Problem, Scale-up |
| 국세청 「주택임대소득 분리과세」 안내 | 2019 전면 과세~ | 연 2천만원 이하 주택임대소득도 분리과세 대상 → 임대인 소득·증빙 관리 필요 | Solution §수입관리 |
| 행정안전부/지자체 「임대주택 분쟁·하자 민원」 일반 통계 | 연간 | 보증금 반환·수선의무·관리비 분쟁이 주택 임대차 민원의 큰 비중 | Problem §하자/분쟁 |
| 한국부동산원 「공동주택 관리비 공개」 제도 | 2023 확대~ | 50세대 이상 관리비 공개 의무화 흐름 → 소액·소규모도 투명성 요구 증가 | Solution §관리비 |
| 중소벤처기업부/창업진흥원 「창업기업 동향」 | 연간 | 프롭테크·생활밀착 SaaS 창업 증가 추세 | Scale-up §시장 |

> ⚠️ 위 표의 수치 중 정확한 연도별 절대값이 본문에 들어갈 경우, 해당 문장에 각주로 출처·연도를 달고, 절대값이 미확정이면 `[추정]`을 병기한다. 본 PoC 단계 제안서는 시장 **구조와 방향성**(임차가구 비중·임대차 중심 시장·신고제/과세로 인한 행정부담 증가)을 근거로 삼으며, 세부 시장 금액(TAM/SAM/SOM)은 공개 통계로 역산한 `[추정]`임을 본문에 명시한다.

## 2. 각주 (제안서 본문 `[^n]`과 1:1)

[^1]: **통계청 「인구주택총조사」** (2020 기준 공표). 전국 일반가구의 점유형태 중 임차(전세+보증부월세+월세+사글세) 가구 비중 약 38~39%. https://kostat.go.kr / KOSIS(https://kosis.kr).
[^2]: **국토교통부 「주택 임대차 신고제」** (2021-06 시행). 보증금 6,000만원 또는 월세 30만원을 초과하는 주택 임대차 계약은 30일 내 신고 의무. https://www.molit.go.kr · 부동산거래관리시스템 https://rtms.molit.go.kr.
[^3]: **한국부동산원·통계청 「전월세 임대차 통계/거래량」** (월간 공표). 임대차 시장은 전세·월세 거래가 상시 발생하는 임대차 중심 구조. https://www.reb.or.kr · KOSIS.
[^4]: **국세청 「주택임대소득 과세」** (2019 귀속분부터 연 2천만원 이하 분리과세 시행). 주택임대소득 신고·증빙 관리 의무. https://www.nts.go.kr.
[^5]: **국토교통부/한국부동산원 「공동주택 관리비 비교·공개」** (2023 관리비 공개 대상 확대). 관리비 투명성 요구 확산. https://www.k-apt.go.kr.
[^6]: **중소벤처기업부·창업진흥원 「창업기업 동향」** (연간). 생활밀착·프롭테크 분야 창업 증가. https://www.mss.go.kr · https://www.kised.or.kr.

## 3. 데이터 정직성 메모

- 본 폴더의 출처는 모두 **공개 1차 자료(공공통계·부처 제도)** 우선이다.
- 임대지기 제안서의 시장규모(TAM/SAM/SOM) 금액은 공개 통계(임차가구 수 × 가정 ARPU)로 역산한 **`[추정]`**이며, 본문에서 추정임을 명시한다.
- 경쟁사 가격·기능 비교표는 2026-06 기준 **공개 정보 기반 일반화**이며, 실제 요금제는 변동될 수 있어 `[재확인 필요]`로 둔다.
