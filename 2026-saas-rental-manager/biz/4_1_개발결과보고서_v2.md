## v2 신규/심화 산출물

### 실 알고리즘 3종 (입력→계산→의사결정)
1. **연체 위험 스코어링** — 임차인별 6개월 지연일수·미납·빈도·잔여계약을 가중합산해 0~100점 + A/B/C/D 등급 + 권고 산출. 등급 분포 차트 + 산출 근거 모달(가중 요소·월별 납부 이력). 결정적(동일 입력→동일 점수).
2. **적정 임대료 추정** — 동일 건물 입주 세대 단가(원/㎡) 중앙값 + 면적 보정 + 시장가 50% 가중 → 추정가·갭(%)·권고. 세대 상세에서 즉시 확인.
3. **수익률 분석** — 건물별 EGI·OpEx·NOI·Cap rate·자기자본·대출이자·세전현금흐름·ROI 계산 + Cap rate/ROI 비교 차트.

### 외부 시스템 통합 3건 (CDN 라이브러리 제외)
1. **카카오 알림톡 mock 발송** — 독촉·사전안내·계약체결 알림 → 알림 센터에 채널·수신자·템플릿·시각·상태 로그 누적.
2. **CSV 입출력** — 수납내역 내보내기(BOM 포함 한글 안전) + 은행 입금내역 붙여넣기 → 세대명·대상월 매칭 자동 입금대사(수납 처리).
3. **전자계약 서명 흐름 mock** — 동의 체크 → 캔버스 서명 입력(마우스/터치) → 서명 이미지 계약서 PDF 삽입 + 알림톡 발송.

### 다중 사용자·다중 테넌트
- 3역할(임대인·관리소장·임차인) 권한 분기, 여러 건물/포트폴리오, 역할 전환 스위처.

## 2. 구현/제작 범위

- **뷰: 임대인 9 / 관리소장 5 / 임차인 3종** (역할 합산 중복 제외 핵심 화면 다수). 각 뷰는 표시가 아니라 인터랙션 가능.
  - 임대인: 포트폴리오 대시보드 · 건물·세대 · 청구·수납 · 연체 관리 · 계약·전자서명 · 하자보수 · 수익률 분석 · 세무·정산 · 알림 센터.
  - 관리소장: 운영 현황(할 일 큐) · 세대 · 수납 · 하자보수 · 알림.
  - 임차인: 내 임대 현황 · 청구·납부 · 하자 신고.
- **워크플로 3개+:** ① 미납→수납→영수증 PDF, ② 연체 진단→사전 안내 알림톡 발송, ③ 계약→전자서명→PDF 반영→알림톡, ④ CSV 입금내역→자동 입금대사→수납 갱신, ⑤ 임차인 하자 신고→임대인 알림톡→관리소장 진행/완료.
- **실 동작(토스트 mock 아님):** 연체 스코어 실 계산, 수익률 실 계산, 캔버스 서명 실 드로잉·PDF 삽입, CSV 실 생성/파싱, localStorage 지속, jsPDF+html2canvas 실 PDF, Chart.js 실 차트.
- 로그인 없이 3역할 전환·시연, 키 불요·오프라인 구동.

```bash
# 1) 앱: 브라우저로 단일 파일 열기 (서버 불요)
open projects/rental-manager/v2.html      # ?reset 로 시드 초기화 / v1.html 도 그대로 열림

# 2) 캡처 재현 (선택)
cd projects/rental-manager
npm i -D playwright && npx playwright install chromium
node capture.mjs                  # 기본 v2.html → biz/captures/v2, biz/captures/mobile/v2
TARGET=v1.html node capture.mjs   # v1 도 캡처 가능
rm -rf node_modules package*.json # 산출물 커밋 금지
```
