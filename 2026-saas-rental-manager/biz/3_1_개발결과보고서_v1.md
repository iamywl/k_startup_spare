## 2. 구현/제작 범위

- **뷰 5종:** ① 수입 대시보드 ② 호실·임차인 관리 ③ 월세 수납 추적 ④ 계약 관리 ⑤ 하자보수 요청. 각 뷰는 표시가 아니라 인터랙션 가능(등록·배정·수납·독촉·PDF·사진·상태전이).
- **다단계 워크플로:** 호실 등록 → 임차인 배정 → 월세 청구(자동 생성) → 수납 체크 → 미납 시 독촉(mock 로그) → 영수증 PDF. 5단계 이상.
- **실 동작(토스트 mock 아님):** localStorage 지속(새로고침 유지), jsPDF+html2canvas 실 PDF 다운로드, 사진 실 업로드/미리보기, Chart.js 실 차트.
- 로그인 없이 즉시 사용(데모 시드), 키 불요·오프라인 구동.

## 3. 환경

| 항목 | 값 |
|:---|:---|
| OS | macOS (Darwin 24.6.0) |
| 런타임 | 단일 HTML, Chromium(Playwright)·일반 브라우저 |
| 라이브러리 | Tailwind(CDN), Chart.js 4.4.1, jsPDF 2.5.1, html2canvas 1.4.1, Pretendard |
| 캡처 | Playwright chromium — PC 1280×800 / 모바일 390×844(isMobile,hasTouch) |

## 4. 실행/구동 방법

```bash
# 1) 앱: 브라우저로 단일 파일 열기 (서버 불요)
open projects/rental-manager/v1.html      # ?reset 로 시드 초기화

# 2) 캡처 재현 (선택)
cd projects/rental-manager
npm i -D playwright && npx playwright install chromium
node capture.mjs          # PC→biz/captures/v1, 모바일→biz/captures/mobile/v1
rm -rf node_modules       # 산출물 커밋 금지
```
