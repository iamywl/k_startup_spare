# 과업지시서 v1 — D2C 통합 CDP / ROAS 어트리뷰션 SaaS

> 외주 위탁 분: **데이터 시각화 디자인 시스템** (가정 시나리오). SW 구현 본체는 자체 수행.

## 1. 과업의 개요

### 과업의 목적
1) 광고·매출 데이터를 직관적으로 보여주는 차트·KPI 카드의 시각 디자인을 외주로 확보한다.
2) 제안서([1_제안서.md](./1_제안서.md))의 "한국 광고 매체 1급 연동 + SMB 가격대" 가치를 데이터 시각화 품질로 뒷받침한다.

### 과업의 범위
1) 외주가 수행할 업무
   - 차트 라이브러리(Recharts) 위 적용할 색상·범례·툴팁 가이드
   - KPI 카드(매출·ROAS·전환·CAC) 4종 시각 시안
   - 어트리뷰션 흐름도 일러스트
2) 범위 외 (Out of Scope)
   - 차트 코드 구현 (자체 수행)
   - 데이터 모델·스키마 설계
   - 광고 API 연동 코드

## 2. 과업의 세부 내용

- 타깃 사용자: D2C 브랜드 마케터(20~40대), 데이터 리터러시 중급
- 디자인 톤: 정보 밀도 높고 차분, 강조색은 ROAS 양/음으로만 사용
- 차트: 라인(추이), 바(매체별 비교), 도넛(기여도), 깔때기(전환)
- 다크 모드 옵션 제공
