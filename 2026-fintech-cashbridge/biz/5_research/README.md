# 5_research — 근거자료 (목표 출처 1,000+)

> `CLAUDE.md` §2.5·§2.6. 실제 존재하는 출처만(날조·중복 금지). 진행 중이면 현재/목표 수치 명시.

**수집 현황: 9 / 1,000 (정직 표기).** 본 사이클은 제안서의 핵심 주장만 우선 인용했다. 대량 출처·고객 인터뷰·파일럿 데이터는 후속 사이클에서 누적하며, 미검증 출처를 날조해 채우지 않는다. 제안서 본문의 모든 미검증 수치(요율·한도·LTV·CAC·손실률·시장규모 등)는 `[추정]`으로 표기했고, 추정값과 공식 수치를 한 문장에 섞지 않았다.

조사 우선순위는 공공 통계 포털 → 정부 부처 → 공공기관·국책연구원 → 학술 → 국제기구 순(§2.5)이며, 아래 목록은 그 우선순위대로 배치했다.
