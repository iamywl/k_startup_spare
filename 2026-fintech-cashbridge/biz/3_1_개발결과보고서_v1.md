| 캡처 | Playwright(Chromium), PC 뷰포트 1280×800 / 모바일 390×844(deviceScaleFactor 2) |

## 4. 실행/구동 방법

1. 브라우저에서 `projects/cashbridge/v1.html` 을 `file://` 로 연다(빌드·설치 불요).
2. 상단/사이드 역할 전환(사업주·심사역·관리자)으로 RBAC 화면 분기 확인.
3. 선정산 신청 → 4단계 스텝퍼(한도→심사→약정→실행) 진행.
4. 리포트 뷰에서 한글 PDF 발행·CSV 입출력, 연동 뷰에서 테스트 webhook 발사.
5. 새로고침 → localStorage 상태 복원 확인.
6. (재캡처) `projects/cashbridge/` 에서 Playwright 설치 후 `node capture.mjs` → `biz/captures/v1/`·`biz/captures/mobile/v1/` 생성(스크립트는 사용 후 의존성 제거).
