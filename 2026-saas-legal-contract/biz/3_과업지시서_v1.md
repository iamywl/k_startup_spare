# 과업지시서 v1 — 리걸체크 계약서 검토·생성 시제품

> 발주자가 외주 용역사에 「리걸체크」 v1 시제품 개발을 위탁하기 위한 지시서.
> 작성 근거: [`2_개발계획서.md`](./2_개발계획서.md) §WBS의 외주 구분 행(#2~#8) → 본 §1 과업의 범위.

---

## 1. 과업의 개요

### 과업의 목적

1) **직접 목적:** 계약서 텍스트를 입력하면 룰기반 엔진이 독소조항을 탐지·하이라이트하고, 조항별 수정안을 제시하며, 표준계약서를 생성하고, 한글 검토 리포트 PDF를 발행하는 **단일 HTML 웹 시제품 1식**을 제작한다.
2) **사업 연결:** 본 시제품은 [`1_제안서.md`](./1_제안서.md)의 솔루션(§2)을 실증하는 데모로, 창업동아리 지원사업 심사·투자 시연·초기 사용자 검증에 사용한다.

### 과업의 범위

**In Scope (외주사 수행)**
- 룰기반 위험탐지 엔진(10종 독소조항 룰셋·정량 가중·0~100 위험점수)
- 분석 결과 뷰(KPI·위험분포 막대차트·종합 게이지·본문 하이라이트)
- 수정 제안 뷰(조항별 "왜 위험/어떻게 수정" + 채택 토글)
- 표준계약서 생성기(용역·공급·임대차 템플릿 + 항목 입력 + 미리보기)
- 검토 리포트 PDF 발행(jsPDF, **한글 깨짐 없음**)
- 검토 이력 localStorage 지속(새로고침 후 유지·재열람·PDF 재발행)
- 반응형 레이아웃(데스크톱 사이드바 ↔ 모바일 바텀탭) + 디자인 토큰

**Out of Scope (외주사 미수행)**
- 서버·DB·계정/로그인·결제 시스템 구축(시제품은 클라이언트 단일 HTML)
- 실 LLM/외부 AI API 연동(v1은 결정론적 룰엔진)
- 실 전자서명·법적 효력 보증·실제 법률 자문
- 캡처 스크립트 작성·문서 작성(발주자 자체 수행)

## 2. 과업의 세부 내용

### 2.1 기능 사양

| 기능 | 사양 |
|:---|:---|
| 입력 | 텍스트 붙여넣기, `.txt` 업로드(FileReader, UTF-8), 예시 불러오기 |
| 룰셋 | 최소 10종: 과도한 위약금·일방해지·자동갱신·일방편의 관할·포괄적 IP귀속·무제한 책임·불리한 대금조건·비대칭 비밀유지·과도한 경업금지·일방적 계약변경 |
| 점수 | 등급계수(고1.4·중1.0·저0.6) × 룰 가중치 합, 0~100 정규화, 60↑ 고위험/30↑ 중위험 |
| 하이라이트 | 매칭 조항을 본문에서 등급별 색 `<mark>` 처리, 클릭 시 해당 제안으로 이동 |
| 수정 제안 | 룰별 "왜 위험한가/이렇게 고치세요" 텍스트 + 채택 토글 + 전체 채택 |
| 표준계약서 | 용역·공급·임대차 3종 템플릿, 발주/수급·금액·기간 입력 반영, 미리보기 |
| PDF | jsPDF로 검토 리포트 생성, **한글 100% 보존**(캔버스 텍스트 렌더 후 이미지 삽입 등), 다단계 분할 페이지 |
| 이력 | localStorage 저장, KPI(총검토·평균점수·고위험수·최근일), 재열람·PDF 재발행 |

### 2.2 UI/UX 사양

- 디자인 톤: [디자인_애플_토스.md](../../design-guide/디자인_애플_토스.md) — 브랜드 단일 포인트(신뢰 인디고), 위계는 여백·타이포, 잉크는 hue 섞인 부드러운 회색, WCAG AA 대비.
- 데스크톱: [디자인_PC.md](../../design-guide/디자인_PC.md) — 좌측 사이드바 240~256px + 본문 `min-w-0`, KPI 4열·차트 2열.
- 모바일: [디자인_모바일.md](../../design-guide/디자인_모바일.md) — 사이드바 숨김·**바텀 탭바**(5목적지), KPI 2열 스택, 차트 고정높이, 탭 ≥44px, 가로 overflow 0.
- 화면 흐름(5뷰): 입력 → 분석 → 제안 → 표준계약서 생성 → 이력. 워크플로 1개 이상(입력→분석→하이라이트→채택→PDF).

### 2.3 라이브러리·의존성

| 항목 | 버전 | 비고 |
|:---|:---|:---|
| Chart.js | 4.x (CDN) | 위험분포·게이지 |
| jsPDF | 2.5.x (CDN) | 리포트 PDF |
| 외부 API 키 | **없음** | 전부 클라이언트, 키 하드코딩 금지 |

### 2.4 품질·성능 기준 (정량)

- 좁은 화면(390px) 가로 overflow **0px**, 콘솔 pageerror **0건**.
- 주요 탭/버튼 터치 타깃 **≥44px**.
- 예시 계약서 분석 응답 **1초 이내**(클라이언트 룰 매칭).
- PDF 한글 깨짐 **0건**(육안 확인).
- 새로고침 후 검토 이력 **유지율 100%**.
