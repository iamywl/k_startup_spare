## v3 신규/심화 산출물

### 1) 고급 분석 알고리즘 심화
- **TF-IDF 코사인 유사도 diff:** 표준조항 코퍼스를 빌드해 IDF(역문서빈도)를 산출, 내 계약 조항과 룰별 균형 표준 문구를 각각 **어절 + 한글 bigram TF-IDF 벡터**로 변환해 **코사인 유사도**로 비교한다. 희소·핵심 보호어에 IDF 가중이 실려 v2 Jaccard보다 독소/누락/정합 분류가 정교하다. 유사도 ≥0.30=정합, <0.30=독소, 핵심 보호조항(책임제한·비밀유지·불가항력) 부재 시 누락. 평균 코사인=표준 정합도(%).
- **등급 캘리브레이션:** 고위험 조항 ≥3 시 임계를 하향(고위험 52/중위험 26), 거의 없을 시 상향(68/36)으로 점수→등급 매핑을 동적 보정한다. 분석 화면 종합 게이지에 적용 임계가 표기된다.
- **협상 시뮬레이터:** 조항별 ‘표준대로 수정’ 토글 시 해당 조항 위험 가중을 표준 수준(18%)으로 낮춘 위험점수를 즉시 재계산하고, before/after 막대 차트와 등급 변화(예: 100 → 73, −27점)를 표시한다. 선택한 수정안을 케이스에 일괄 채택할 수 있다.
- **법령/판례 매칭 DB:** 22룰 각각에 KO/EN 법령·판례 근거를 내장해 협상 카드·검토 리포트 PDF에 근거를 노출한다.

### 2) 다중 사용자/다중 테넌트 (v2 계승 + 심화)
- **3역할 RBAC**(의뢰인·검토자·관리자) 스위처. 관리자 전용으로 **감사 로그**·**관리자 리포트**·CSV/JSON 입출력을 노출하고, 검토자는 작업 큐·코멘트 작성자 분기를 갖는다.
- **실시간 팀 협업 알림:** 검토자가 코멘트를 등록하면 `collab.comment.created` webhook과 팀원(2명) 알림 로그가 협업 화면에 실제로 적재된다.

### 3) 외부 시스템 통합 (CDN·라이브러리 제외)
- **① 전자서명 OAuth 2.0 + webhook 콜백 mock:** `GET .../oauth/authorize?...state=...` → `CALLBACK ...?code=...&state=...`(state 검증) → `POST /oauth/token`(access_token 교환) 핸드셰이크 로그. 연결 후에만 서명 요청 활성, 서명 완료 시 `WEBHOOK esign.completed` 콜백으로 케이스 완료 전이. 알림톡 발송 로그 병행.
- **② 팀 협업 알림 webhook:** 코멘트 작성 이벤트를 webhook + 팀 알림 로그로 외부 전파(mock).
- **③ CSV/JSON 외부 입출력(v2 계승):** 포트폴리오 CSV(BOM·따옴표 이스케이프)·JSON 백업 실제 다운로드, JSON 가져오기 병합.

### 4) 다국어(i18n)·거버넌스
- **KO/EN i18n:** 사전 기반 토글로 전 UI(사이드바·바텀탭·룰 카테고리·법령 근거·표준 조항·차트 라벨·PDF)가 즉시 전환되고 언어 선택이 localStorage에 기억된다.
- **감사 로그 + 계약 버전관리:** 모든 케이스/코멘트/서명/버전 이벤트를 시간순 기록하고 유형으로 필터한다. 협상 반영본을 버전 스냅샷으로 저장해 버전 간 조항 추가·삭제·위험점수 변화를 추적한다.

## 2. 구현·제작 범위

- **13개 뷰** 실동작: ① 케이스 보드 ② 계약서 입력 ③ 위험 분석(등급 캘리브레이션) ④ 표준 대비 diff(TF-IDF 코사인) ⑤ 협상 우선순위 ⑥ 협상 시뮬레이터 ⑦ 표준계약서 생성 ⑧ 협업 코멘트(팀 알림) ⑨ 전자서명(OAuth) ⑩ 버전 관리 ⑪ 관리자 리포트 ⑫ 감사 로그 ⑬ 검토 이력.
- **워크플로 3개:** (a) 입력→분석→diff→협상 채택/시뮬레이션→OAuth 서명(검토·협상·체결 파이프라인), (b) 검토자 코멘트→팀 알림→해결→단계 전이(실시간 협업), (c) 케이스 포트폴리오→버전 스냅샷→감사 로그→CSV/JSON 입출력(거버넌스/백업).
- **고급 알고리즘:** TF-IDF 코사인 유사도 diff, 등급 캘리브레이션, 협상 시뮬레이터(조건 변경→재계산), 22룰 가중치 모델.
- **다중 사용자:** 3역할 RBAC, 케이스 보드, 팀 협업 알림.
- **외부 통합:** 전자서명 OAuth 2.0 + webhook 콜백 mock, 팀 협업 알림 webhook, CSV/JSON 입출력.
- **다국어·거버넌스:** KO/EN i18n, 감사 로그, 계약 버전관리.

## 4. 실행/구동 방법

```bash
# 1) 앱 실행 — 브라우저로 v3.html 열기 (빌드 불필요)
open projects/legal-check/v3.html

# 2) 캡처 재현 (선택)
cd projects/legal-check
npm i -D playwright && npx playwright install chromium   # 또는 글로벌 chromium 사용
node capture.mjs            # PC→biz/captures/v3, 모바일→biz/captures/mobile/v3
rm -rf node_modules package*.json
```

앱 사용: ‘데모 케이스 채우기’로 보드 채움 → ‘계약서 입력 → 예시 불러오기 → 조항 분석 시작’ → 분석(등급 캘리브레이션)·표준 diff(TF-IDF 코사인)·협상 우선순위 확인 → **협상 시뮬레이터**에서 조항 토글로 점수 재계산 → 검토자로 코멘트 작성(팀 알림 webhook) → 의뢰인으로 전자서명 OAuth 연결→서명→webhook 완료 → 버전 스냅샷 저장 → 관리자로 감사 로그·CSV/JSON 확인 → **KO/EN 토글**로 전 UI 전환.
