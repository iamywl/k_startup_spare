## v3 신규/심화 산출물

### A. 고급 알고리즘 3종 (v2의 4종 위에 추가)

1. **질병 위험 예측 (로지스틱 회귀)** — 피처 `[연령 정규화, BCS 이탈, 최근증상(90일), 일정지연, 견종 플래그]`에 질환별 가중 `w`·절편 `b`를 적용해 `z=w·x+b`, `sigmoid(z)`로 **질환별 발병 확률**을 산출하고 내림차순 정렬·최우선 위험을 권고한다. 견·묘 종별 질환 모델(예: 견 — 척추 IVDD·슬개골·심장·치주·대사질환 / 묘 — 만성 신장질환·당뇨·FLUTD·구강). 콜라(웰시코기)의 척추 IVDD 59.3%가 최우선으로 산출됨(견종 위험 플래그 가중). (PC 08)
2. **성장곡선 percentile** — 품종 표준 성체 체중(평균·표준편차)과 월령 성숙 시그모이드로 기대치를 만들고, z-score를 정규분포 CDF로 변환해 **percentile**과 표준 미만/초과 밴드를 판정. 실측/기대 두 곡선을 차트로 표시. (PC 08 우측, 모바일 26)
3. **약물 상호작용 체크** — 현재 투약 약물을 클래스로 매핑한 뒤 룰 기반으로 충돌 페어를 탐지(major/moderate/minor)하고 설명을 제시. 콜라의 **NSAIDs + 스테로이드 = major(병용 금기)**가 탐지됨. 약물 추가 시 즉시 재점검. (PC 09)

### B. 병원 admin 역할 — 4역할 RBAC 완성

- v2의 보호자(인디고)·수의사(틸)·보험사(퍼플)에 **병원 admin(주황)** 추가. 운영 대시보드(미확정/확정 예약·환자 수·기록 매출·EMR 상태·위험 상위 환자), 예약·진료 캘린더(확정·캘린더 동기화), 환자 차트를 제공. 역할 전환 시 내비게이션·뷰·테마·바텀탭이 즉시 분기. (PC 17·18)

### C. 외부 통합 확장 4종 (v2의 4종 위에 추가, 모두 키 0 mock)

- **EMR 양방향 연동** — 병원 admin이 EMR(iVET) 연결 → 진료기록 1건을 pull → **보호자 타임라인·수의사 기록에 동시 반영**, 연결 상태·최근 동기화·가져온 기록 수·발송 로그 기록. (PC 17)
- **다보험사 청구 API** — 청구 작성 시 3개 보험사(삼성화재·메리츠·DB) 중 선택해 보험사 API(mock)로 라우팅·접수번호 발급. (PC 10·12)
- **웨어러블 OAuth 2.0** — 인가 흐름 mock으로 access_token 발급·로그 기록, **인가 후에만** 활동량 동기화 허용(미인가 시 안내). (PC 12)
- **캘린더 동기화** — 병원 admin이 외부 캘린더(구글) 연결 → 예약·접종 일정 push 동기화, 동기화 일정 수·최근 시각 표시. (PC 18)

### D. i18n (KO/EN)

- 언어 토글을 사이드바·드로어·모바일 상단바에 노출, UI 셸·내비게이션·역할·섹션·핵심 라벨을 사전 기반 전환하고 localStorage로 언어를 지속(새로고침 후 유지). (PC 23·24, 모바일 23·24)

## 2. 구현/제작 범위

- **뷰 21종**: 보호자 10(홈·반려동물·건강기록·스케줄·건강리포트·**질병 위험 예측**·병원예약·보험청구·데이터연동·알림센터) + 수의사 4 + 보험사 3 + **병원 admin 3**(운영 대시보드·예약 캘린더·환자 차트).
- **워크플로 3개+**: ① 펫 등록 → 프로토콜 자동 시드 → D-day → 알림톡, ② 진료기록(수의사/EMR pull) → 보호자 타임라인 반영 → 알림, ③ 진료 선택 → 다보험사 청구서 → 제출 → 심사 → 승인/반려, (+ ④ 웨어러블 OAuth 인가 → 활동량 동기화, ⑤ 병원 EMR·캘린더 연동).
- **실 알고리즘 7종**(v2 4 + v3 3) + **외부 통합 mock 8종**(v2 4 + v3 4).
- **실동작 액션**: 사진 업로드·미리보기, localStorage 영속(역할·언어·연동상태·로그·청구 포함), 주기 자동 계산·완료 재계산, jsPDF PDF 3종(canvas 렌더, 한글 깨짐 0), CSV/JSON 입출력, 차트 5종(추세·활동량·리스크 도넛·체중·**성장곡선**).
- **반응형**: 모바일(≤767px) 오프캔버스 드로어(역할 전환·언어) + 역할별 바텀탭, PC(≥768px) 사이드바 256px. 인증 없이 즉시 사용(데모 시드: 보호자A 콜라·나비, 보호자B 보리).

```bash
# 1) 데모 실행 (브라우저에서 직접 열기)

# 2) 캡처 재현 (Playwright)
cd 2026-saas-pet-health/projects/pet-health
npm i -D playwright && npx playwright install chromium
node capture.mjs        # PC → biz/captures/v3, 모바일 → biz/captures/mobile/v3
rm -rf node_modules package.json package-lock.json   # 캡처 후 정리
```
