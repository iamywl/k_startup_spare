## 기술 스택

| 계층 | 선택 |
|:---|:---|
| 시제품 | 단일 HTML + Vanilla JS (오프라인 자체완결) |
| 차트 | Chart.js (CDN) |
| PDF | jsPDF (CDN) + canvas 렌더(한글 폰트 깨짐 회피) |
| 저장 | localStorage(시제품) → 정식: 동기화 DB |
| 정식(v2+) | Node.js API + PostgreSQL + 푸시/알림톡(키 없으면 mock) |
| 캡처 | Playwright (PC 1280×800 / 모바일 390×844), 실행 후 node_modules 삭제 |
