## 3. 솔루션 성숙도

| 영역 | 단계 | 근거 |
|---|---|---|
| 범용 챗봇·생성 도구 (B2C) | 확장기 | 한국인 44.5% 생성형 AI 경험, 전년比 +11.2%p (과기정통부 2025)[^15] |
| 엔터프라이즈 LLM Ops·문서 AI (B2B) | 상용 초기→확장기 | 업스테이지 2024년 매출 700억원대[^16] |
| 버티컬 AI SaaS (법률·의료·금융·교육) | PoC~상용 초기 | Gartner: PoC 실패율 高, CIO 자체개발 축소·상용 패키지 선호[^14] |
| 멀티모달·에이전트 | PoC | 트웰브랩스·디노티시아 원천기술 단계[^8][^10] |

## 4. 사업 모델
B2B SaaS 구독 / API 토큰 과금 / Freemium→Pro / 캐릭터 IAP / 온프렘 sLLM 라이선스.

## 5. White Space

| White Space | 비어있는 이유 |
|---|---|
| 한국어 규제산업(의료·법률·세무) 특화 RAG SaaS | 도메인 데이터 라이선싱·인허가 장벽으로 외산 LLM 진입 어려움 |
| 중소 제조·현장직 음성 AI 비서 | 사무직 경험률 71.9% vs 생산직 급감[^15] — 모바일·음성 UX 부재 |
| AI 에이전트 ROI·관측(Observability) 툴 | Gartner 지적 PoC 실패의 핵심 원인[^14] |
| 온디바이스·sLLM 배포 미들웨어 | 노타·디노티시아가 인프라 단, 응용 SaaS 레이어 비어있음 |
| 공공·교육 도메인 한국어 평가·안전성 SaaS | KISA·NIA 표준화 초기, 상용 사업자 부재 |
