## 3. 환경
- OS/브라우저: macOS · Chromium(Playwright 자동화)
- 언어/구성: 순수 HTML/CSS/JavaScript 단일 파일(빌드리스). 외부 의존성 = jsPDF 2.5.1 (CDN, 부재 시 폴백)
- 뷰포트: PC 1280×900 / 모바일 390×844(deviceScaleFactor 2, isMobile, hasTouch)
- 저장: 브라우저 localStorage (`safeon-v1`)

## 4. 실행/구동 방법
1. `projects/site-safety/index.html` 을 브라우저로 연다(서버 불필요, `file://` 가능).
2. 최초 진입 시 데모 현장·근로자·교육 데이터가 자동 시드된다.
3. 사이드바(PC)/바텀탭·드로어(모바일)로 8개 뷰 이동. 위험성평가·TBM·아차사고는 위저드로 끝까지 진행 가능.
4. 재캡처: `cd projects/site-safety && node capture.mjs` → `biz/captures/v1`·`biz/captures/mobile/v1` 갱신.
